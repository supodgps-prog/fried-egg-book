# Lesson Plan: วิธีทอดไข่ดาว

## ระยะเวลา

60 นาที กว่า 123 789

## กลุ่มเป้าหมาย

ผู้เริ่มต้นทำอาหาร

## Learning Outcomes

- อธิบายอุปกรณ์และวัตถุดิบได้
- ทำตามขั้นตอนการทอดไข่ดาวได้
- วิเคราะห์ปัญหาเบื้องต้นได้

## แผนการสอน

| เวลา | กิจกรรม |
|---:|---|
| 0–10 นาที | แนะนำวัตถุประสงค์ |
| 10–20 นาที | อธิบายอุปกรณ์และวัตถุดิบ |
| 20–40 นาที | สาธิตการทอดไข่ดาว |
| 40–50 นาที | ให้ผู้เรียนทดลอง |
| 50–60 นาที | สรุปและตอบคำถาม |
