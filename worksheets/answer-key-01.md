# Answer Key 01

1. เพื่อป้องกันเปลือกไข่ตกลงกระทะ และตรวจสอบคุณภาพไข่ก่อนทอด
2. น้ำมันยังไม่ร้อน หรือกระทะไม่เหมาะสม
3. ประมาณ 2 นาที
4. ตัวอย่าง Flowchart: เริ่ม → เตรียมอุปกรณ์ → ตั้งกระทะ → ใส่น้ำมัน → ใส่ไข่ → ตรวจสอบความสุก → ตักเสิร์ฟ → จบ
5. Input = ไข่ น้ำมัน ความร้อน  
   Process = การทอด  
   Output = ไข่ดาว
