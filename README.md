# Fried Egg Course

สื่อการสอนตัวอย่างเรื่อง “วิธีทอดไข่ดาว”  
ใช้สำหรับฝึกสร้างหนังสือ เอกสารประกอบการสอน สไลด์ แบบฝึกหัด Assignment และเฉลย ด้วย Markdown
![รูปที่ 1 ไข่ดาวที่ทอดเสร็จ](./images/chapter01/egg01.jpg)

## Learning Outcomes

หลังเรียนจบ ผู้เรียนสามารถ

1. เตรียมอุปกรณ์และวัตถุดิบได้ถูกต้อง
2. อธิบายขั้นตอนการทอดไข่ดาวได้
3. ควบคุมเวลาและความร้อนในการทอดได้
4. ประเมินผลลัพธ์ของไข่ดาวได้
5. วิเคราะห์ปัญหาที่เกิดขึ้นระหว่างการทอดได้
