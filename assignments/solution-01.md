
# Solution 01

## Input

- ไข่ไก่
- น้ำมัน
- ความร้อน
- เวลา

## Process

- ตั้งกระทะ
- ใส่น้ำมัน
- รอความร้อน
- ทอดไข่
- ตรวจสอบความสุก

## Output

- ไข่ดาว

## Sensor

- ตา: ดูสีและความสุก
- จมูก: ตรวจกลิ่นไหม้
- หู: ฟังเสียงน้ำมัน
- เวลา: ใช้นาฬิกาจับเวลา

## Control Action

- เพิ่มหรือลดไฟ
- พลิกไข่
- เพิ่มเวลาทอด
- ตักขึ้นเมื่อสุก

## Error

| Error | สาเหตุ | วิธีแก้ |
|---|---|---|
| ไข่ไหม้ | ไฟแรงเกินไป | ลดไฟ |
| ไข่ไม่สุก | เวลาน้อยเกินไป | ทอดต่อ |
| ไข่ติดกระทะ | น้ำมันไม่พอ | เพิ่มน้ำมัน |
